How to Receive Messages

To receive messages from your contact form, you need to set up server-side processing. Here are the options:

Option 1: Formspree Service

Free form submission service that emails you the messages.

<form action="https://formspree.io/f/mgvzyjqa" method="POST">
  <!-- Your form fields -->
</form>

Option 2: PHP Script

Create a PHP file on your server to process the form.

<?php
$name = $_POST['name'];
$email = $_POST['email'];
// Process and email the data
mail('you@example.com', 'New Contact', $message);
?>

Option 3: Node.js Backend

Set up a Node.js server with Express to handle submissions.

app.post('/contact', (req, res) => {
  const {name, email} = req.body;
  // Send email or save to database
});
